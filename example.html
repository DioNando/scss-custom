Framework Structure

The framework is organized following the ITCSS (Inverted Triangle CSS) methodology for optimal scalability.

1. Abstracts

Variables, functions, mixins, and placeholders. No CSS output.

2. Base

Reset, typography, and base element styles.

3. Utilities

Single-purpose utility classes for common properties.

4. Components

Reusable component styles (buttons, cards, forms).

Getting Started

Quick steps to start using the framework in your project.

Installation

  1. Ensure you have a SCSS compiler set up in your project
  2. Import scss/main.scss in your compilation pipeline
  3. The compiled CSS will be available as css/main.css
  4. Link the CSS file in your HTML: <link rel="stylesheet" href="path/to/css/main.css">

Customization

Modify the design system by editing variables in scss/abstracts/_variables.scss:

// Colors Map
$colors: (
    primary: #2563eb,
    secondary: #64748b,
    // ... more colors
);
